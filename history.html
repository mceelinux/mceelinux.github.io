A bit of history

There hasn't always been ways of running Minecraft Education on Linux, but there's always someone trying to.

29th March 2023

First documentation for Minecraft Education in Wine

techguy16 creates a GitHub Gist (that's still maintained to this day!) to allow users to install Minecraft Education in Wine, although this wasn't finished until later.

26th June 2024

MCEELinux project established

techguy16 establishes the MCEELinux project, a centralised organisation that aims to find methods to run Minecraft Education on Linux.
